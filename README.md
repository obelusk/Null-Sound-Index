# Null Sound Index

Part of the **Null Suite** — free tools to download, use, and modify.

Browse and preview sound files in a folder without leaving the app. Null Sound Index is a lightweight desktop tool for quickly auditioning sound effects, music, and audio clips — see a waveform, hit play, and move on, no separate media player required.

## Features

- Folder tree browser with expandable subfolders
- Instant playback with waveform preview
- Drag and drop sound files directly into other apps (e.g. a DAW, video editor, or file explorer)
- Loop playback
- Minimal Mode — a small, focused player window showing just the waveform, file name, and controls
- Customizable accent color and keyboard shortcuts
- Set a default folder to open automatically on launch
- Built-in update checker

## Installation

1. Go to the [Releases page](https://github.com/obelusk/Null-Sound-Index/releases).
2. Under the latest release, download **`Null-Sound-Index-Setup-X.X.X.exe`** (ignore the `.blockmap` and `.yml` files — those are only used by the in-app updater).
3. Run the downloaded installer.
4. **You'll likely see a Windows SmartScreen warning** ("Windows protected your PC") — this happens because the installer isn't signed with a paid commercial certificate, not because anything is wrong. Click **More info**, then **Run anyway** to continue.
5. Follow the install wizard — you can choose the install location and whether to create Desktop/Start Menu shortcuts.
6. Launch **Null Sound Index** from the Start Menu or Desktop shortcut.

## Updating

Null Sound Index checks for updates automatically on launch, and you can manually check anytime via **File > Check for Updates…**. If a new version is available, you'll be prompted to download and install it — declining is always an option.

## Keyboard Shortcuts

All shortcuts are customizable in **File > User Settings**. Defaults:

| Action | Key |
|---|---|
| Play / Pause | Space |
| Next file | ↓ |
| Previous file | ↑ |
| Next folder | → |
| Previous folder | ← |
| Toggle Loop | L |
| Toggle Minimal Mode | M |
